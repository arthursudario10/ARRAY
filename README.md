# Manipulação de Objetos e Arrays em JavaScript

Este projeto demonstra a utilização de arrays de objetos em JavaScript, acessando propriedades específicas e exibindo informações no console e em alertas.

---

# Objetivo

Criar um array contendo informações de alunos, como nome, idade e disciplina preferida, e acessar os dados utilizando índices e propriedades dos objetos.

---

# Estrutura Utilizada

O programa utiliza:

- Arrays
- Objetos
- `console.log()`
- `alert()`

---

# Funcionalidade

O sistema armazena informações de vários alunos em um array e exibe a disciplina preferida de um aluno específico.

---

# Código Desenvolvido

```javascript
const alunos = [
    {name: "gustavo", idade: "16", disciplinaPreferida: "filosofia"},
    {name: "mayara", idade: "16", disciplinaPreferida: "matematica"},
    {name: "livia", idade: "16", disciplinaPreferida: "geografia"},
    {name: "anna", idade: "16", disciplinaPreferida: "portugues"},
    {name: "arthur", idade: "17", disciplinaPreferida: "educação física"}
];

console.log(alunos[0].disciplinaPreferida);

alert("A disciplina preferida desse aluno é: " + alunos[0].disciplinaPreferida);
````
